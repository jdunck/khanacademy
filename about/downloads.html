Offline Server

For an offline capable version of the site for Windows, visit the download page. Unzip it and click on run.bat, which will start up the server and load the local version of the Khan Academy homepage in your browser. All the exercises should work at this point.

Next go to the download_scripts directory and click on the playlist(s) you want to download (download_ALL.bat will get everything). When you're done, the videos you've downloaded will work when you click on them in the browser.

Once you've downloaded your playlists, feel free to distribute them, as long as it's not done on a for-profit basis. You could burn a DVD or USB stick and send it to someone without internet access, or order the equivalent from our store.

DVD Store

If someone you know has a computer running Windows and a DVD drive, but no internet access, they can still use the Khan Academy offline videos and exercises on DVD. This is the same as you can download here but can be helpful if your internet connection is slow, you don't know how to burn a DVD, or you would just like someone to do the downloading/burning/mailing for you.

To help us gauge interest in this, please leave a comment telling us which DVD you would like to purchase. The prices will probably be around $10 per DVD, or the 7 DVD set for $24. That should be around our breakeven point for burning and mailing them, but if it's off we'll adjust the price to make it that way.
